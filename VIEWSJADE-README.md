# Views with Jade
================================

## Usage
- have nodejs setup
- Install hapi
  npm install hapi

#### Modules
--------------------------------
- npm install joi
- npm install jade

### Directories needed
--------------------------------
- public // contains images dir with pngs
- views // contains hello.jade file

##### Test
---------------------------------
- node filename.js
- running on localhost:8888
- url: localhost:8888/hello/firstname/lastname?mood=(choose happy, sad, neutral).
